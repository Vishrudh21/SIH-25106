Under the Hood

Architecture & Data Backbone

Stack Overview

Responsive React/PWA frontend, NestJS API, PostgreSQL core database, Redis cache, MinIO/S3 storage, Keycloak auth, and workflow / notification microservices orchestrated via queues.

Integrations cover campus SSO, calendar sync (ICS/Google), SMTP/SMS, and analytics dashboards (Metabase/Grafana). 

Frontend → API Gateway → Services (Profiles, Opportunities, Matching, Workflow)
          ↘ Redis Cache / PostgreSQL / Object Storage
Workflow → Notifications → Email / SMS / Calendar
Observability → Logs, Metrics, Alerts
